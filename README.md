# rfacun's `dwmblocks`
This custom build of `dwmblocks` is part of [my dotfiles](https://github.com/rfacun/dotfiles).
Apart from the scripts that I added inside `blocks.h`, the program is 99% the
same as the original [here](https://github.com/torrinfail/dwmblocks).

## Install
```
git clone https://github.com/rfacun/dwmblocks.git
cd dwmblocks
sudo make clean install
```
